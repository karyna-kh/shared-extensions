# Apollo.io FlowRunner Extension

A comprehensive sales intelligence and lead enrichment integration that connects Apollo's prospecting database with automated workflows. This extension enables AI-powered lead qualification, contact data verification, and systematic engagement processes for building targeted prospect lists and managing outreach campaigns.

## Ideal Use Cases

- Building AI agents for automated lead qualification and prospecting
- Enriching CRM data with comprehensive professional contact details
- Creating targeted prospect lists using advanced search filters
- Automating follow-up workflows and scheduling outreach activities
- Conducting market research and competitor landscape analysis
- Managing account-based marketing campaigns with systematic engagement
- Validating contact information before outreach campaigns

## List of Actions

- Add Contacts to Sequence
- Create Task
- Enrich Organization
- Enrich People
- Enrich Person
- Get a List of Users
- Get Organization Job Postings
- People Search
- Search Contacts
- Search Organizations
- Search Sequences
