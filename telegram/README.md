# Telegram FlowRunner Extension

Comprehensive Telegram Bot API integration for automated messaging, group management, and webhook handling. Enables full bot functionality including sending messages, media files, and location data, along with real-time event monitoring for building interactive bot workflows.

## Ideal Use Cases

- Building automated customer support and chatbots
- Implementing notification and alert systems via Telegram
- Managing groups and channels with automated moderation
- Distributing news and content to multiple chats
- Creating interactive bot workflows with commands and callbacks
- Setting up real-time monitoring for messages and events
- Automating file and media sharing processes
- Broadcasting announcements to multiple channels
- Collecting user feedback and engagement data
- Forwarding important messages between chats
